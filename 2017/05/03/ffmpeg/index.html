在某些情况下，您可能不知道要加入水印的视频的确切尺寸。 幸运的是，有一些变量可以用来更好地定位水印，这取决于视频的大小。 这些变量包括：

  • main_h - 视频的高度
  • main_w - 视频的宽度
  • overlay_h - 重叠广告的高度
  • overlay_w - 重叠式广告的宽度
